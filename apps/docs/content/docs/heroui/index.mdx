## Prerequisites

Before you start, make sure you have the following prerequisites integrated into your project:

- [Better Auth](https://www.better-auth.com/docs/installation)
- [HeroUI v3](https://v3.heroui.com/docs/quick-start)
- [Sonner](https://sonner.emilkowal.ski/getting-started)

## Installation

<Steps>
<Step>
### Install the library

Install the `@better-auth-ui/heroui` library using your preferred package manager.

```package-install
@better-auth-ui/heroui@latest
```

</Step>
<Step>
### Configure TailwindCSS

Add the following `@import` to your global css.

```css title="styles/globals.css"
@import "@better-auth-ui/heroui/styles";
```
</Step>
</Steps>
